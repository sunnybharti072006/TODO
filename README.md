# 📝 To-Do List App (Kotlin + Jetpack Compose + Room DB)

A simple and beginner-friendly To-Do List Android application built using **Kotlin**, **Jetpack Compose**, and **Room Database**.
This project is designed with a **minimal structure**, where core logic is handled in a single file (`Todo.kt`) for simplicity and learning.

---

## 🚀 Features

* ➕ **Add Task**
* 🗑️ **Delete Task**
* ✅ **Mark as Completed**

---

## 💡 Why This Project is Good

* 🎨 **Simple UI with Jetpack Compose** (No XML)
* 📦 **All logic in one file (Todo.kt)** – Easy to understand
* 🔄 **Basic CRUD operations using Room DB**
* 👶 **Perfect for Beginners**

---
